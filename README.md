# English Dictionary

English Dictionary is a native Android application that allows users to browse, search, add, view, and delete English words with their Turkish meanings. The app uses a local SQLite database, including a preloaded dictionary database from assets, and provides a simple RecyclerView-based interface for quick word lookup and vocabulary management.

## Tech Stack

- Java
- Android SDK
- SQLite
- RecyclerView
- AppCompat
- Material Components
- ConstraintLayout
- Gradle Kotlin DSL
